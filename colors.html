Accessibility

All color combinations must meet WCAG 2.1 contrast ratio requirements to ensure content is readable for all users.

Contrast Requirements

AA Normal 4.5:1

Required for body text, labels, and all text smaller than 18px (or 14px bold).

AA Large 3:1

Acceptable for large text: headings 18px+ regular or 14px+ bold weight.

AAA 7:1

Enhanced accessibility level for improved readability and inclusive design.

Best Practices

Use semantic tokens first

Only reach for raw palette values when building custom components that need specific shade control.

Test in both themes

Always verify color choices render well in both light and dark mode before shipping.

Don't rely on color alone

Pair color indicators with icons, text, or patterns for users with color vision deficiencies.

Check contrast ratios

Use browser dev tools or accessibility checkers to verify WCAG compliance.
