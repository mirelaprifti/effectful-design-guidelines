Nested Radius

When nesting rounded elements, the inner radius should be smaller to maintain visual harmony. Use this formula:

inner radius = outer radius − padding

When padding ≥ outer radius, use rounded-none or rounded-sm

Radius + Shadows

Box shadows automatically follow the border radius curve. Larger shadows pair well with a larger corner radius for a cohesive elevated appearance.

Design rationale: Larger shadows suggest more visual elevation—elements appear "higher" off the surface. Higher elements benefit from softer, more rounded corners to maintain visual harmony.

Guidelines

Scale Together

Larger shadows suggest more elevation. Pair with a larger corner radius to maintain proportion.

Combine with Border

For subtle elevation, use shadow-sm with a border for definition.

Best Practices

✓

Consider the Content

Larger containers benefit from a larger corner radius; small, dense UI elements work better with a smaller corner radius.

✓

Avoid Mixing Too Many Values

Stick to 2-3 radius values per component to maintain visual consistency.

✓

Be Consistent Within Groups

All buttons in a button group should share the same radius.
