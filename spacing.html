When to Use Px vs Rem

Use Rem

For spacing that should scale with user font preferences

  • Component padding and margins
  • Gap between elements
  • Section spacing
  • Container padding

Use px

For fixed dimensions requiring precise control

  • Border widths
  • Icon sizes
  • Shadows and outlines
  • Fixed-size decorative elements

Best Practices

✓

Stick to the Scale

Always use values from the spacing scale rather than arbitrary pixel values.

✓

Use Consistent Increments

Jump by consistent steps (e.g., 2 → 4 → 6 → 8) rather than random values.

✓

Larger Spacing for Sections

Use mb-16 to mb-20 between major page sections.

✓

Prefer Gap Over Margin

Use CSS gap in flex/grid layouts instead of margins on children.
