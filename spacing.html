Best Practices

  • Stick to the scale. Always use values from the spacing scale rather than arbitrary pixel values.
  • Use consistent increments. Jump by consistent steps (e.g., 2 → 4 → 6 → 8) rather than random values.
  • Larger spacing for sections. Use mb-16 to mb-20 between major page sections.
  • Prefer gap over margin. Use CSS gap in flex/grid layouts instead of margins on children.
  • Responsive spacing. Use responsive prefixes (md:p-8) to increase spacing on larger screens.
  • Group related elements. Use tighter spacing (2-4) within groups and larger spacing (6-8) between groups.
