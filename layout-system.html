Best Practices

  • Mobile-first always. Write base styles for mobile, then add responsive prefixes for larger screens.
  • Use container consistently. Wrap page content in container mx-auto with responsive horizontal padding.
  • Constrain content width. Use max-w-5xl or similar to prevent text from spanning too wide on large screens.
  • Prefer Grid for 2D layouts. Use CSS Grid when you need both row and column control. Use Flexbox for single-axis alignment.
  • Test at all breakpoints. Verify layouts work at every breakpoint, not just desktop and mobile.
  • Avoid fixed widths. Use percentage-based or fractional widths (w-1/2, flex-1) for responsive elements.
