Best Practices

✓

Maintain Hierarchy

Limit to 3-4 text sizes per page for a clear visual hierarchy.

✓

Balance Headings

Use text-balance to prevent awkward line breaks in titles.

✓

Accessibility First

Ensure sufficient color contrast (4.5:1 for body text, 3:1 for large text).

✓

Consistent Line Height

Use tighter line height for headings, relaxed for body paragraphs.
