Best Practices

  • Use text-balance for headings to prevent awkward line breaks
  • Maintain a clear hierarchy with 3-4 text sizes per page
  • Reserve display sizes for hero sections and marketing
  • Use font-mono exclusively for code and technical content
  • Ensure sufficient color contrast for accessibility
