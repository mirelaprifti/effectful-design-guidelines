Best Practices

  • Limit heading levels. Use no more than 3-4 heading sizes per page to maintain clear hierarchy.
  • Body text at 14-16px. Use text-sm (14px) or text-base (16px) for readable body content.
  • Use weight for emphasis. Prefer font-medium or font-semibold over font size changes when adding emphasis.
  • Consistent line heights. Use leading-relaxed for body text, leading-tight for headings.
  • Monospace for code. Always use font-mono for code snippets, technical values, and data displays.
  • Respect max-width. Limit line length to 65-75 characters for optimal readability using max-w-prose.
